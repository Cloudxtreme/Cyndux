Digital Media
Graphic files vs Movie Files
A graphic file is just a picture, and therefore has a single frame, while a movie uses multiple frames, to create the illusion of motion when played right.
Multimedia
The use of multiple mediums, such as video and sound to make a film.
Social Network
A dedicated website or other application which enables users to communicate with each other by posting information, comments, messages, images, etc.
Image Resolution
The amount of detail a raster image holds. The more the resolution, the larger the detail, amount of pixels and file size the image has.
Search Engine
A program, such as Google, that searches for and identifies items in a database that cois related to the keywords or characters specified by the user, used especially for finding particular sites on the World Wide Web.
Vector vs Raster Images
A raster image uses pixels, small squares on a screen to create a picture, while vector images use simple dot to dot co-ordinates to draw mulitple lines, fill in spaces, to achieve the same goal.

Hardware
Peripheral Devices
Devices (monitors, mouse, keyboards) that input and output information provided by a computer, that is simple enough for an end user to understand.
CPU Components
The arithmetic logic unit is the part of the CPU that, as its name implies, carries out the mathematical functions of addition, subtraction, multiplication and division. The control unit is another fundamental part of the CPU. Essentially, it regulates the flow of information through the processor. More components of a CPU that are vital to its operation are the registers, which are very small memory locations that are responsible for holding the data that is to be processed.
